# v0.0.163 — csv

Expand `csv` module (src/vm/csv_mod.zig).

## What's in

- `QUOTE_STRINGS`, `QUOTE_NOTNULL` constants (4, 5)
- `csv.Error` exception class (inherits Exception)
- `reader` returns an object with `line_num` and `dialect` attrs; supports `skipinitialspace`
- `writer` supports all quoting modes, custom `lineterminator`, `escapechar`
- `Dialect` class with `delimiter`, `quotechar`, `doublequote`, `skipinitialspace`, `lineterminator`, `quoting`
- Built-in dialects: `excel`, `excel-tab`, `unix`
- `register_dialect`, `unregister_dialect`, `list_dialects`, `get_dialect`
- `DictReader` object with `fieldnames`, `restval`, `restkey` support
- `DictWriter` with `extrasaction='raise'/'ignore'`
- `field_size_limit()`
- `Sniffer` with `sniff()` (delimiter detection) and `has_header()`
