# v0.0.189

Implement comprehensive threading fixture (189_threading).

## Changes

- `src/vm/threading_mod.zig`:
  - Lock.acquire gains `blocking` kwarg support
  - Thread.start() defers execution to a pending queue instead of running inline
  - Thread.join() drains the pending queue before finalizing the thread
  - Condition.wait() releases the lock, drains pending threads (enabling producer/consumer patterns), then re-acquires
  - BoundedSemaphore gets its own class with bounds-checked release (raises ValueError)
  - current_thread() returns the active thread instance during target execution
  - active_count() counts live threads + main thread
  - enumerate() returns main thread + all live threads
  - main_thread() always returns the MainThread instance
- `src/vm/interp.zig`: add `threading_bounded_sem_class`, `threading_live_threads`, `threading_pending_threads`, `threading_current_thread` fields
- `tests/fixtures/189_threading.*`: fixture from goipy
