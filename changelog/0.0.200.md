# v0.0.200

Implement socket module (fixture 200).

## Changes

- `src/vm/socket_mod.zig`: new module with socket class (bind, listen,
  accept, connect, connect_ex, sendall, send, recv, sendto, recvfrom,
  close, setsockopt, getsockopt, getsockname, getpeername, setblocking,
  getblocking, settimeout, gettimeout, shutdown, fileno); module
  functions socketpair, create_connection, gethostname, getaddrinfo,
  inet_aton/ntoa/pton/ntop, ntohs/ntohl/htons/htonl, gethostbyname,
  getservbyname, getprotobyname, setdefaulttimeout/getdefaulttimeout;
  error/timeout/gaierror/herror exception classes; AF_INET/AF_INET6/
  AF_UNIX/SOCK_STREAM/SOCK_DGRAM/IPPROTO_TCP/IPPROTO_UDP/SOL_SOCKET/
  SO_REUSEADDR/SHUT_*/INADDR_ANY constants.  recv() tries MSG_DONTWAIT
  first; on EAGAIN it drains pending Python threads (the
  start→client→join pattern) then does a blocking read so TCP loopback
  tests complete without real OS threads.
- `src/vm/threading_mod.zig`: make runPendingThreads pub so socket_mod
  can call it from recv/accept.
- `src/vm/interp.zig`: add socket_module, socket_class, socket_error_class,
  socket_timeout_class, socket_gaierror_class, socket_herror_class fields
  and register "socket" in getBuiltinModule.
- `tests/fixtures/200_*`: fixture from goipy.
