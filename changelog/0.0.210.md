# v0.0.210

Extend base64 module to pass fixture 210.

## Changes

- `src/vm/base64_mod.zig`: rewrite with kwarg support via BuiltinKwFnPtr; b64encode/b64decode
  altchars and validate kwargs; strict padding validation in b64decode (raise ValueError for
  non-multiple-of-4 input, matching CPython); urlsafe_b64decode auto-padding; b32decode/
  b32hexdecode/b16decode casefold kwarg; encodebytes/decodebytes 57-byte chunk + newline
  formatting; b32hexencode/b32hexdecode with B32HEX alphabet; b85encode/b85decode RFC 1924
  with correct partial-group padding ('~' value 84); a85encode/a85decode ASCII85 with adobe
  and foldspaces kwargs; standard_b64encode/standard_b64decode aliases.
- `tests/fixtures/210_*`: fixture from goipy.
