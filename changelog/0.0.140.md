# 0.0.140 — 2026-04-26

### Added

- `operator.abs`, `operator.inv` / `operator.invert`, `operator.is_` /
  `operator.is_not`, `operator.concat`, `operator.length_hint`,
  `operator.countOf`, `operator.indexOf`.
- The full `i*` in-place family: `iadd`, `iconcat`, `isub`, `imul`,
  `itruediv`, `ifloordiv`, `imod`, `ipow`, `ilshift`, `irshift`,
  `iand`, `ior`, `ixor`. `iadd` / `iconcat` mutate the destination
  list in place so `result is target` still holds; immutable types
  fall back to plain add.
- `__dunder__` aliases (`__add__`, `__mul__`, `__lt__`, `__abs__`,
  `__inv__`, `__not__`, `__concat__`, `__contains__`, `__getitem__`,
  `__iadd__`, ...) that mirror every public name.
