# 0.0.174

Fixture 174: logging module.

- Named loggers now propagate log records to root logger handlers (matching
  Python's default `propagate=True` behavior)
- Effective level resolution walks up the logger hierarchy instead of
  falling back to DEBUG unconditionally
- `Formatter` with `%(levelname)s:%(name)s:%(message)s` format works
- `StreamHandler(sys.stdout)` routes log output to captured stdout
