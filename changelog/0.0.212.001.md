# v0.0.212.1

Patch: fix CI on Linux (link_libc) and rename changelog files to zero-padded format.

## Changes

- `build.zig`: add `link_libc = true` to the zag module so Linux/glibc builds
  compile without "dependency on libc must be explicitly specified" errors.
- `changelog/`: rename all files from `0.0.N.md` to `0.0.NNN.md` (zero-padded
  to three digits) so directory listings sort correctly by version number.
  Patch files follow the same pattern: `0.0.N.M.md` → `0.0.NNN.MMM.md`.
