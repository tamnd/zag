# v0.0.162 ‚Äî compression.zstd

Add `compression.zstd` module (src/vm/zstd_mod.zig).

## What's in

- `zstd.compress(data)` / `zstd.decompress(data)` ‚Äî module-level functions
- `ZstdCompressor` ‚Äî buffers data, emits frame on `flush()` or `compress(data, FLUSH_FRAME)`
- `ZstdDecompressor` ‚Äî streaming; supports chunked input, `max_length`, `eof`, `unused_data`
- `ZstdDict(data, is_raw=True)` ‚Äî dict wrapper
- `ZstdFile(path, mode)` ‚Äî read/write/seek/tell/peek/readline; context manager
- `zstd.open(path, mode)` ‚Äî binary and text modes
- `ZstdError` exception class
- Constants: `COMPRESSION_LEVEL_DEFAULT`, `zstd_version`, `zstd_version_info`
- `ZstdCompressor.CONTINUE`, `FLUSH_BLOCK`, `FLUSH_FRAME` class constants
