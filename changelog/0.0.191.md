# v0.0.191

Implement multiprocessing.shared_memory module (fixture 191).

## Changes

- `src/vm/shared_memory_mod.zig`: new module with SharedMemory and
  ShareableList. Uses an interp-level name→data registry (sm_registry)
  so named attach and cross-process writes work within the same
  interpreter run. SharedMemory exposes a buf proxy for byte-level
  read/write. ShareableList supports iteration, count, index, and
  arbitrary Python value types.
- `src/vm/interp.zig`: add sm_registry, sm_shm_class, sm_buf_class,
  sm_sl_class fields; register multiprocessing.shared_memory module
- `tests/fixtures/191_shared_memory.*`: fixture from goipy
