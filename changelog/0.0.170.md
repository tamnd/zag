# v0.0.170 — secrets

Expand `secrets` module (src/vm/secrets_mod.zig).

## What's in

- `DEFAULT_ENTROPY` constant (32)
- `token_bytes(n)`, `token_hex(n)`, `token_urlsafe(n)` raise `ValueError("negative argument not allowed")` for negative n
- `randbelow` error message matches CPython: "Upper bound must be positive."
- `randbits` error message: "number of bits must be non-negative"
- `choice([])` IndexError message: "Cannot choose from an empty sequence"
