# 0.0.145 — 2026-04-26

### Added

- `tempfile` rewrite covering the surface for fixtures: `gettempdir`,
  `gettempdirb`, `gettempprefix`, `gettempprefixb`, the mutable
  module-level `tempdir`, `mktemp`, `mkstemp`, `mkdtemp`, and the
  `TemporaryFile` / `NamedTemporaryFile` / `SpooledTemporaryFile`
  constructors. The three file shapes share one buffer-backed file
  class with `read`/`write`/`seek`/`tell`/`close`/`writable`/`readable`/
  `seekable`/`__enter__`/`__exit__`/`rollover`. `NamedTemporaryFile`
  creates an actual on-disk file so `os.path.isfile(f.name)` holds
  during the context, and honors `delete=`. `TemporaryDirectory`
  picks up `suffix`/`prefix`/`dir`/`delete` kwargs.
- `os.close` (no-op) so the synthetic fds returned by `mkstemp`
  round-trip cleanly.
