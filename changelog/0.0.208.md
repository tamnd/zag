# v0.0.208

Add `mailbox` module (fixture 208).

Implements in-memory `mbox` and `Maildir` stores with `add`, `remove`,
`discard`, `clear`, `keys`, `__contains__`, `get_string`, `get_bytes`,
and `__getitem__` returning an `mboxMessage` that parses headers on
demand. Also adds `mailbox.Error`, `NoSuchMailboxError`, and
`FormatError` exception classes.
