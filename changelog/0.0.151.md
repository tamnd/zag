# 0.0.151 — 2026-04-26

### Added

- `copyreg` module: `dispatch_table` (shared dict on interp),
  `pickle(type, reducer[, ctor])`, `constructor(callable)`,
  `add_extension(module, name, code)`, `remove_extension`,
  `clear_extension_cache`, `__newobj__`, `__newobj_ex__`. Validation
  matches CPython: code ≤ 0 raises `ValueError`; conflicting
  (module, name) ↔ code pairs raise `ValueError`; non-callable args
  raise `TypeError`.

### Fixed

- `Value.equals`: same-tag heap types (`builtin_fn`, `function`,
  `class`, `module`, etc.) now fall back to identity comparison so
  they work as dict keys (required by `copyreg.dispatch_table[tuple]`).
- `containsOp` for dicts: previously short-circuited on any non-str
  key with `false`; now performs a linear scan using `valuesEqual` so
  `tuple in copyreg.dispatch_table` works correctly.
