# 0.0.175

Fixture 175: uuid module.

- `uuid.uuid5(namespace, name)` — SHA-1 based deterministic UUID
- `uuid.uuid3(namespace, name)` — MD5 based deterministic UUID
- `uuid.uuid4()` — random version 4 UUID
- `uuid.UUID(str)` — construct from canonical string form
- `uuid.NAMESPACE_DNS`, `uuid.NAMESPACE_URL`, `uuid.NAMESPACE_OID`,
  `uuid.NAMESPACE_X500` — standard namespace constants
- UUID instances expose `.version` attribute
