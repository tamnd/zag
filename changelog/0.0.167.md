# v0.0.167 — plistlib

Add `plistlib` module (src/vm/plistlib_mod.zig).

## What's in

- `dumps(obj, fmt=FMT_XML, sort_keys=True)` / `loads(data, fmt=None, aware_datetime=False)`
- `dump(obj, fp)` / `load(fp)` — file-like object support
- `FMT_XML` (1) and `FMT_BINARY` (2) — `PlistFormat` enum with repr `<PlistFormat.FMT_XML: 1>`
- `UID(n)` class with `.data` attr and `repr` → `UID(42)`
- `InvalidFileException` inherits Exception
- XML plist: string/integer/real/bool/data/date/array/dict; XML entity escaping
- Binary plist (`bplist00`): full bplist00 encoding/decoding; bool/int/float/date/data/string/array/dict/UID
- UTF-16 binary strings for non-ASCII content (`héllo`)
- Datetime round-trip (Apple epoch = 2001-01-01); naive and aware
- Integer boundary values: 0..255 (1-byte), ..65535 (2-byte), ..2^32-1 (4-byte), signed 8-byte for negative/large
- Auto-detect format on `loads` (checks `bplist00` prefix)
- Tuple serialized as array
- bytearray serialized as data
