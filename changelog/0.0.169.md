# v0.0.169 — hmac

Expand `hmac` module (src/vm/hmac_mod.zig).

## What's in

- `sha3_256` and `sha3_512` as supported `digestmod` values — both by name string and by hashlib callable
- `copy()` method — deep-copies the running HMAC state so the original keeps accumulating independently
- Block sizes corrected for SHA3: sha3_256 uses 136-byte rate, sha3_512 uses 72-byte rate
- Buffer size for key/pad arrays raised to 200 bytes to cover all supported block sizes
