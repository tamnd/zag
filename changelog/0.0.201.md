# v0.0.201

Implement ssl module (fixture 201).

## Changes

- `src/vm/ssl_mod.zig`: new module with fake-TLS implementation —
  SSLContext (PROTOCOL_TLS_CLIENT/SERVER, check_hostname, verify_mode,
  minimum_version, maximum_version, load_cert_chain, set_ciphers,
  set_alpn_protocols, wrap_socket); SSLSocket (recv, send, sendall,
  close, version, cipher, getpeercert, selected_alpn_protocol,
  compression, pending, fileno); exception hierarchy (SSLError→OSError,
  SSLZeroReturnError, SSLWantReadError, SSLWantWriteError, SSLEOFError,
  SSLCertVerificationError, CertificateError); TLSVersion class with
  TLSv1_2/TLSv1_3 singletons; create_default_context,
  get_default_verify_paths, PEM_cert_to_DER_cert, DER_cert_to_PEM_cert,
  RAND_bytes, RAND_status, get_server_certificate; constants
  PROTOCOL_TLS_CLIENT/SERVER, CERT_NONE/OPTIONAL/REQUIRED, HAS_ALPN,
  HAS_SNI, HAS_TLSv1_3, OPENSSL_VERSION, OP_NO_*.  No real TLS
  handshake: wrap_socket returns immediately, data flows as plaintext;
  get_server_certificate connects+closes TCP so server recv gets EOF,
  then returns the cert stored by load_cert_chain.
- `src/vm/interp.zig`: add ssl_module, ssl_context_class,
  ssl_socket_class, ssl_error_class and six sub-exception class fields,
  ssl_tls_version_class, ssl_default_verify_paths_class,
  ssl_last_cert_pem, ssl_tlsv1_2, ssl_tlsv1_3; register "ssl" in
  getBuiltinModule.
- `tests/fixtures/201_*`: fixture from goipy.
