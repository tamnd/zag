# 0.0.177

Fixture 177: contextlib module.

- `ExitStack.__exit__` now forwards the active exception info to each
  registered exit callback; previously it always called callbacks with
  `(None, None, None)` so exceptions were never suppressed
- `ExitStack` now returns `True` when any registered exit suppresses
  the exception (enabling `ExitStack + suppress` to work correctly)
- `contextmanager`, `suppress`, `nullcontext`, `closing` already worked
