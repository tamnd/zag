# v0.0.205

Implement mmap module (fixture 205).

## Changes

- `src/vm/mmap_mod.zig`: new module with ACCESS_DEFAULT/READ/WRITE/COPY
  constants, PROT_READ/WRITE, MAP_SHARED/PRIVATE, MADV_SEQUENTIAL, PAGESIZE
  and ALLOCATIONGRANULARITY (from getpagesize()); mmap class with __init__
  supporting anonymous (fd=-1, MAP_PRIVATE|MAP_ANON) and file-backed
  (MAP_SHARED) mappings; methods: read, write, write_byte, seek, tell, size,
  close, flush, find, rfind, readline, madvise, __getitem__ (int and slice),
  __setitem__ (slice), __enter__, __exit__; closed attribute; ACCESS_READ
  protection enforced in write/write_byte; resolves virtual os fds to real OS
  file descriptors via interp.os_fd_table.
- `src/vm/interp.zig`: add mmap_module and mmap_class fields; register "mmap"
  in getBuiltinModule.
- `tests/fixtures/205_*`: fixture from goipy.
