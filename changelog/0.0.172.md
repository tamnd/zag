# 0.0.172

Fixture 172: subprocess module.

- `subprocess.CalledProcessError` is now a proper exception class with a
  `returncode` attribute; `except subprocess.CalledProcessError as e:` works
- `subprocess.run()` now accepts the `input` keyword argument for piping bytes
  to the child's stdin
- `subprocess.CompletedProcess` class is exposed on the module
