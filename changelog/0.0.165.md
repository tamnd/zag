# v0.0.165 тАФ tomllib

Add `tomllib` module (src/vm/tomllib_mod.zig).

## What's in

- `loads(s)` тАФ parse TOML string to dict
- `load(fp)` тАФ read from binary file-like object and parse
- `TOMLDecodeError` (inherits `ValueError`) тАФ raised on parse errors
- Full value type support: strings (basic, literal, multiline, all escapes), integers (decimal, hex, octal, binary, underscores), floats (including `inf`, `-inf`, `nan`), booleans, datetimes (offset datetime, local datetime, local date, local time with fractional seconds), arrays, inline tables
- Tables `[header]` and array of tables `[[header]]`
- Dotted keys (`a.b.c = 1`) and quoted keys
- Comments
- Duplicate key/table detection

Also fixed float `repr()`: use scientific notation when exponent is outside `[-4, 16)`, matching CPython's output for values like `6.626e-34`.
