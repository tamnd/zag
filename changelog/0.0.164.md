# v0.0.164 — configparser

Expand `configparser` module (src/vm/configparser_mod.zig).

## What's in

- `DEFAULTSECT`, `MAX_INTERPOLATION_DEPTH` constants
- `BasicInterpolation` — `%(key)s` substitution within same section, recursive up to depth 10
- `ExtendedInterpolation` — `${section:key}` and `${key}` substitution
- `SectionProxy` — `cfg['section']` returns proxy with `__getitem__`, `__contains__`, `keys`, `items`
- `DuplicateOptionError`, `MissingSectionHeaderError`, `InterpolationMissingOptionError` exception classes
- `strict` mode raises on duplicate options/sections
- `allow_no_value` — keys without `=` stored as `None`
- `inline_comment_prefixes`, custom `delimiters`, `comment_prefixes`
- `read_file(fp)` — reads from file-like object
- `read_dict(d)` — loads from dict of section→dict mappings
- `defaults()` — returns Dict of DEFAULT key/values
- `items(section)` — returns list of `(key, val)` tuples
- `get(section, option, fallback=...)` — fallback kwarg support
- `getint`, `getfloat`, `getboolean` — all support `fallback` kwarg
- `get(raw=True)` — skip interpolation
- `write(space_around_delimiters=False)` — compact output
- `BOOLEAN_STATES` dict attribute on `ConfigParser` class
