# v0.0.193

Implement concurrent.interpreters module (fixture 193).

## Changes

- `src/vm/concurrent_interpreters_mod.zig`: new module with Interpreter,
  Queue, create(), list_all(), get_main(), get_current(), create_queue().
  Interpreter.exec() uses a micro-parser for simple statement patterns
  (raise, VAR.METHOD(EXPR)). Interpreter.call() dispatches directly.
  Exception classes (ExecutionFailed, InterpreterNotFoundError,
  QueueEmpty, QueueFull) are registered in builtins on first import.
- `src/vm/interp.zig`: add ci_* fields; cache module to avoid class
  identity mismatch on repeated imports
- `tests/fixtures/193_*`: fixture from goipy
