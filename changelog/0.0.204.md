# v0.0.204

Implement signal module (fixture 204).

## Changes

- `src/vm/signal_mod.zig`: new module with signal constants (SIG_DFL/IGN,
  SIGINT/TERM/USR1/USR2/ALRM and all macOS signals, ITIMER_*, SIG_BLOCK/
  UNBLOCK/SETMASK); strsignal (returns None for out-of-range signum);
  valid_signals (returns list of signals 1..31); default_int_handler
  (raises KeyboardInterrupt); getsignal/signal (module-level handler table
  in g_handlers[65]); raise_signal (calls handler synchronously for
  callables, SIG_IGN is no-op, SIG_DFL sends real signal via libc raise);
  set_wakeup_fd (no-op, returns -1); alarm (libc alarm()); pthread_sigmask
  (libc pthread_sigmask with u32 sigset_t).
- `src/vm/interp.zig`: add signal_module field; register "signal" in
  getBuiltinModule.
- `tests/fixtures/204_*`: fixture from goipy.
