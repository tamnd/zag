# v0.0.190

Implement multiprocessing module (fixture 190).

## Changes

- `src/vm/multiprocessing_mod.zig`: new module with Process, Queue, Pipe,
  Pool, Value, Array, Manager, Lock, Event, Semaphore, BoundedSemaphore,
  Condition, Barrier. Processes use deferred pending queue (same model as
  threading). Barrier.wait() drains pending procs for two-party rendezvous.
  condWait() and condWaitFor() cooperatively run pending procs.
- `src/vm/interp.zig`: add 18 mp_* fields and register `multiprocessing` module
- `tests/fixtures/190_multiprocessing.*`: fixture from goipy
