# 0.0.171

Fixture 171: extended os module.

- `os.makedirs` now accepts the `exist_ok` keyword argument
- Added `os.replace`, `os.link`, `os.truncate`
- Added `os.umask`, `os.getgid`, `os.getegid`, `os.geteuid`, `os.getuid`, `os.getppid`
- Added `os.cpu_count`, `os.strerror`, `os.urandom`
- Added `os.fsencode`, `os.fsdecode`, `os.get_exec_path`
- Added `os.walk` and `os.scandir` (returns `DirEntry` objects with `is_dir`/`is_file`)
- Added low-level fd ops: `os.open`, `os.read`, `os.write`, `os.lseek`, `os.fstat`, `os.dup`, `os.close`
  - `os.dup` shares file position with the original fd (POSIX semantics)
- Added constants: `F_OK`, `R_OK`, `W_OK`, `X_OK`, `O_RDONLY`, `O_WRONLY`, `O_RDWR`, `O_CREAT`, `O_TRUNC`, `SEEK_SET`, `SEEK_CUR`, `SEEK_END`
- `os.stat_result` now supports integer indexing (`st[6]`, `st[4]`, etc.)
- `os.environ` falls back to reading libc `environ` when no env map is provided
