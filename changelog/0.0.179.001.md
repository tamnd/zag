# 0.0.179.1

Patch: align fixture 179 with goipy (was gc, now ctypes).

- `ctypes.CDLL(name)` — load dynamic library stub, `lib.func` returns callable
- `ctypes.sizeof(obj)` — compute size from `_fields_` or scalar type
- `ctypes.Structure` — base class with `_fields_` list of (name, type) pairs
- `ctypes.Union` — like Structure but max-size semantics
- `ctypes.cast(obj, type)` — type-casting stub
- `ctypes.byref(obj)` — byref stub
- `ctypes.pointer(obj)` — pointer wrapper
- Scalar types: `c_int`, `c_long`, `c_uint`, `c_ulong`, `c_longlong`, `c_ulonglong`, `c_size_t`, `c_ssize_t`, `c_float`, `c_double`, `c_char`, `c_char_p`, `c_wchar`, `c_wchar_p`, `c_void_p`, `c_bool`, `c_byte`, `c_ubyte`, `c_short`, `c_ushort`, `py_object`
- `CDLL.in_dll(lib, name)`, `POINTER(type)`, `ARRAY(type, n)` factory
- `WinDLL`, `OleDLL`, `PyDLL` aliases; `windll`, `cdll`, `pydll` loader stubs
