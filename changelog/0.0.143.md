# 0.0.143 — 2026-04-26

### Added

- `stat` module: ST_MODE/INO/DEV/NLINK/UID/GID/SIZE/ATIME/MTIME/CTIME
  tuple indices; the S_IF* file-type bits with S_IFMT/S_IMODE bit
  manipulators; the S_ISDIR/CHR/BLK/REG/FIFO/LNK/SOCK/DOOR/PORT
  predicate functions; full S_IRWX*/S_I[RWX][UGO] permission
  constants and S_ISUID/GID/VTX, plus the legacy S_IREAD/IWRITE/IEXEC
  /ENFMT aliases; BSD/macOS UF_* and SF_* flags; the cross-platform
  FILE_ATTRIBUTE_* set; and `filemode()` that rebuilds the
  `ls -l`-style string with proper setuid/setgid/sticky encoding.
- `os.mkdir`, `os.rmdir`.
