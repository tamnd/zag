# v0.0.192

Implement concurrent.futures module (fixture 192).

## Changes

- `src/vm/concurrent_futures_mod.zig`: new module with
  ThreadPoolExecutor, ProcessPoolExecutor, Future, wait(),
  as_completed(). Deferred-execution model: submit() queues work;
  result()/shutdown()/__exit__ drain synchronously. Future states:
  PENDING → RUNNING → FINISHED or CANCELLED. done_callbacks fire
  on finish or immediately if already done.
- `src/vm/builtins.zig`: add CancelledError exception class; add
  id() builtin returning pointer address as integer
- `src/vm/interp.zig`: add cf_future_class, cf_tpe_class, cf_ppe_class
  fields; register concurrent.futures module
- `tests/fixtures/192_*`: fixture from goipy
