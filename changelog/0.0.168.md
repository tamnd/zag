# v0.0.168 — hashlib

Expand `hashlib` module (src/vm/hashlib_mod.zig).

## What's in

- SHA3 family: `sha3_224`, `sha3_256`, `sha3_384`, `sha3_512` via `std.crypto.hash.sha3`
- SHAKE XOFs: `shake_128`, `shake_256` with `hexdigest(n)` / `digest(n)` taking output length
- Blake2 family: `blake2b` (default 64-byte) and `blake2s` (default 32-byte) with `digest_size=` and `key=` kwargs
- `file_digest(fp, algo)` reads a file-like object and returns a hash object
- `algorithms_available` and `algorithms_guaranteed` sets expanded to include all new algorithms
- Keyword argument support (`usedforsecurity=False`, `digest_size=32`, `key=b"..."`) via `kw_func`
- Block sizes for SHA3: sha3_224=144, sha3_256=136, sha3_384=104, sha3_512=72
