# 0.0.142 — 2026-04-26

### Added

- Full `os.path` surface: constants `sep`, `curdir`, `pardir`, `extsep`,
  `altsep`, `pathsep`, `defpath`, `devnull`, `supports_unicode_filenames`,
  plus `isabs`, `abspath`, `basename`, `dirname`, `split`, `splitext`,
  `splitdrive`, `splitroot`, `join`, `normpath`, `normcase`,
  `commonprefix`, `commonpath`, `relpath`, `expandvars`, `expanduser`,
  `exists`, `lexists`, `isfile`, `isdir`, `islink`, `ismount`,
  `isjunction`, `getsize`, `getmtime`, `getatime`, `getctime`,
  `samefile`, `samestat`, `realpath`. `import os.path` resolves through
  the same builtin lookup as `import os`.
- `os.environ`: a regular dict seeded once from the host env, mutable
  from Python; `os.path.expandvars` reads from it live.
- `os.symlink`, `os.stat` / `os.lstat` (returning a `stat_result`
  instance with `st_mode`, `st_size`, `st_ino`, `st_nlink`, `st_uid`,
  `st_gid`, `st_dev`, `st_atime`, `st_mtime`, `st_ctime`), `os.fspath`.
