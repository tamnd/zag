# v0.0.166 — netrc

Add `netrc` module (src/vm/netrc_mod.zig).

## What's in

- `netrc(path)` — parse a netrc file
- `.hosts` dict — `{host: (login, account, password)}`
- `.macros` dict — `{name: [line, ...]}`
- `.authenticators(host)` — returns `(login, account, password)` tuple or `None`; falls back to `default` entry
- `NetrcParseError` — with `.msg`, `.lineno`, `.filename` attrs
- Quoted values (`"my user"`)
- `macdef` blocks (reads lines until blank line)
- `user` synonym for `login`
- `default` fallback entry
- Comments with `#`
- `__repr__` includes all machine entries
