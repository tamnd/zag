# 0.0.186

Fixture 186: time module extended (goipy 173_time).

- `struct_time[n]` — index access including negative indices
- `time.asctime(st)` — format struct_time as "Thu Jan  1 00:00:00 1970"
- `time.ctime(secs)` — format timestamp as asctime string
- `time.process_time_ns()`, `time.thread_time_ns()` — return int >= 0
- `time.get_clock_info(name)` — object with implementation, monotonic,
  adjustable, resolution attributes
- `CLOCK_REALTIME`, `CLOCK_MONOTONIC`, `CLOCK_PROCESS_CPUTIME_ID`,
  `CLOCK_THREAD_CPUTIME_ID` — integer constants
