# v0.0.161 — tarfile

Add `tarfile` module (src/vm/tarfile_mod.zig).

## What's in

- `tarfile.open(path, mode)` — modes `w`, `r`, `w:gz`, `r:gz`, `w:bz2`, `r:bz2`, `w:xz`, `r:xz`, `r:*` (autodetect)
- `TarFile.addfile(tarinfo, fileobj)` — write entry from BytesIO
- `TarFile.add(path, arcname=...)` — recursively walk a directory
- `TarFile.getnames()`, `getmembers()`, `getmember(name)`
- `TarFile.extractfile(name)` — returns BytesIO
- `TarFile.extractall(path)`
- `TarFile.__iter__` / `__next__`
- `TarInfo(name)` with `isfile()`, `isdir()`
- `tarfile.is_tarfile(path)`
- `TarError`, `ReadError` exception classes
- Constants: `REGTYPE`, `DIRTYPE`
